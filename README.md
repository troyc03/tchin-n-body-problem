# Numerical Approximations Of Solutions to the N-Body Problem

This repository contains all modules for constructing and implementing a mathematical model in Python that simulates the trajectories of an N-body system. The n-body problem is a classical physics challenge of predicting the individual motions of a group of celestial objects interacting through gravity. While the two-body problem is solvable in closed form, systems with three or more bodies are generally impossible to solve analytically, resulting in chaotic, unpredictable long-term behavior that requires numerical approximations.
